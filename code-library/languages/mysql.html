Introduction to MySQL

MySQL is a popular open-source relational database management system. Here are some essential commands and concepts I've learned:

MySQL Commands

Connecting to MySQL

mysql -u username -p

Connects to the MySQL server with the specified username and prompts for password.

Showing Databases

SHOW DATABASES;

Lists all available databases on the server.

Creating a Database

CREATE DATABASE database_name;

Creates a new database with the specified name.

Using a Database

USE database_name;

Selects a database to work with.

Creating Tables

CREATE TABLE table_name ( column1 datatype constraints, column2 datatype constraints, ... );

Creates a new table with specified columns and data types.

Inserting Data

INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);

Adds a new row of data to the table.

Querying Data

SELECT column1, column2 FROM table_name WHERE condition;

Retrieves data from the table based on specified conditions.

Updating Data

UPDATE table_name SET column1 = value1 WHERE condition;

Modifies existing data in the table.

Deleting Data

DELETE FROM table_name WHERE condition;

Removes rows from the table based on conditions.
