Introduction to React

React is a JavaScript library for building user interfaces. Here's what I've learned about React components, hooks, and core concepts:

React Fundamentals

Creating a React App

npx create-react-app my-app

Sets up a new React project with all necessary dependencies.

Functional Components

function MyComponent() { return <div>Hello World</div>; }

The basic building block of React applications.

JSX Syntax

const element = <h1>Hello, world!</h1>;

JSX is a syntax extension that allows HTML-like code in JavaScript.

State with useState Hook

const [count, setCount] = useState(0);

Manages state in functional components. 'count' is the current value, 'setCount' updates it.

Side Effects with useEffect

useEffect(() => { // Side effect code here return () => { // Cleanup code (optional) }; }, [dependencies]);

Handles side effects like data fetching, subscriptions, or manually changing the DOM.

Props

function Welcome(props) { return <h1>Hello, {props.name}</h1>; }

Pass data from parent to child components.

Conditional Rendering

{isLoggedIn ? <LogoutButton /> : <LoginButton />}

Render different components based on conditions.

Lists and Keys

{numbers.map((number) => <li key={number.toString()}>{number}</li> )}

Render lists of elements and assign unique keys for efficient updates.
