Site Plan – Web Development Student Resource Library

1. Site Name

Web Dev Resource Library

I chose this name because it clearly describes the purpose of the website: a helpful place where students can find curated resources related to web development. It’s meant to sound friendly and educational.

2. Site Purpose

This site will serve as a digital library where beginner and intermediate web development students can find valuable learning resources. The site will include links to articles, tutorials, videos, tools, and code examples organized by categories such as HTML, CSS, JavaScript, and Web Tools. The goal is to support and guide learners by giving them easy access to trusted, beginner-friendly materials.

3. Scenarios

  • Where can I find beginner-friendly tutorials for learning CSS Grid?
  • What are some good YouTube channels or websites for learning JavaScript?

4. Color Schema

  • Teal (#1abc9c) – Used for headings and buttons.
  • Dark Blue-Grey (#2c3e50) – Used for navigation bar and footer background.
  • Light Grey (#f4f4f4) – Used as the site background.

5. Typography

  • Segoe UI – Used for general text and body content.
  • Montserrat – (via Google Fonts) will be used for headings and callout sections.
